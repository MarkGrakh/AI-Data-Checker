# 🛡 AI Data Checker — Защита информации перед отправкой в искусственный интеллект

Локальный веб-сервис для проверки текста и файлов на наличие **информации ограниченного доступа** перед тем, как отправить их в публичные ИИ-модели (ChatGPT, GigaChat, YandexGPT и др.).

## 📋 Соответствие законодательству РФ

- ✅ Федеральный закон от 27.07.2006 № 152-ФЗ «О персональных данных»
- ✅ Указ Президента РФ от 10.10.2019 № 490 «О развитии искусственного интеллекта»
- ✅ Приказ ФСТЭК России от 11.04.2025 № 117
- ✅ Постановление Правительства РФ от 01.11.2012 № 1119 (уровни защищённости ПДн)
- ✅ Приказ Роскомнадзора от 05.09.2013 № 996

## 🎯 Возможности

| Функция | Описание |
|---------|----------|
| 🔍 **Проверка текста** | Вставка текста, мгновенный вердикт |
| 📎 **Проверка файлов** | DOCX, PDF, XLSX, TXT (офлайн-парсинг) |
| 👥 **Ролевая модель** | Администратор и пользователь |
| 📋 **Журнал аудита** | Все проверки с деталями |
| 📊 **Статистика** | Метрики по проверкам |
| ⚙️ **Настраиваемые правила** | Запрещённые слова, regexp, фразы |
| 🔒 **Полностью офлайн** | Не требует интернета |

1. Скачайте файл `data_checker_final.html`
2. Откройте в любом браузере
3. Войдите в систему:

| Роль | Логин | Пароль |
|------|-------|--------|
| Администратор | `admin` | `admin123` |
| Пользователь | `user` | `user123` |

## 🔧 Установка правил

Правила проверки соответствуют требованиям 152-ФЗ и включают:

- **Персональные данные**: паспорт, СНИЛС, ИНН, телефон, email, ФИО
- **Спецкатегории**: раса, религия, здоровье, биометрия
- **Тайны**: коммерческая, врачебная, адвокатская, нотариальная
- **Служебная информация**: ДСП, конфиденциально, не подлежит разглашению
- **Финансовые данные**: номера счетов, банковские карты
